## サービス・コンテンツ概要
北洋銀行（北海道地方銀行）の公式スマートフォンアプリのLPページ。
主な機能として「残高照会」「振り込み」「口座開設」の3つをヒーローで訴求。
コンビニATM利用手数料の月1回無料特典、北洋銀行間振込手数料0円などの特典を全面訴求。
24時間利用可能、公共料金支払い、カードローン、投資信託等の管理機能も紹介。

## ターゲットユーザー
- 北洋銀行既存口座保有者（スマホ活用促進）
- 新規口座開設希望者（アプリで口座開設訴求）
- 北海道在住・出身の銀行利用者
- 忙しくて窓口・ATM利用が困難なユーザー

## ヒーローセクション詳細
- **レイアウト:** 2カラム（左：スマホモックアップ + かわうそ、右：テキスト）
- **キャッチコピー:** 「いつでもどこでもスマホでかんたん！」→ 大見出し「北洋銀行アプリ」
- **ビジュアル要素:** リアルなiPhoneモックアップ（実際のアプリUI）、ブルーかわうそキャラ、右上に黄色円形バッジ（コンビニATM無料訴求）
- **CTA:** ヒーロー内CTAボタンなし。機能タグ「残高照会 / 振り込み / 口座開設」を丸枠タグで表示
- **背景:** 純白 rgb(255, 255, 255)

## CTAデザイン
- **形状:** 大型ピル型（角丸の大きいボタン）
- **黄色CTA「詳細はこちらから」:** 黄色背景 + ネイビーテキスト + 円形矢印アイコン付き
- **「ダウンロードはこちら（無料）」:** 黄色テキスト / ブルー背景の固定フッター
- **App Store / Google Playボタン:** 黒背景公式バッジ形式
- **ヒーロータグ:** 白背景・ブルーテキスト・丸枠・等間隔配置

## ナビゲーション

- **雰囲気キーワード:** 親しみやすい、安心・安全、かんたん、無料、便利

## 特徴的なデザイン要素・テクニック
1. **マスコットキャラクター全面活用:** かわうそが全セクションに登場しブランドを統一
2. **黄×青の強コントラスト:** 特典・CTAに黄色でブルー背景から際立たせる
3. **黄色の丸角特典カード:** 3カラムグリッド / マスコットが上部から顔を出すユニーク演出
4. **固定フッターCTA:** スクロール追従のダウンロードバー
5. **タグ型機能表示:** ボタンではなく丸枠タグでアプリ機能を簡潔に見せる
6. **右上黄色円形バッジ:** 最重要特典を吹き出し風の丸バッジで目立たせる
7. **スクロールアニメーション:** animated__fadeIn クラスで要素を動的に登場させる
8. **FAQアコーディオン:** ブルー背景 / 白カード / Qの青丸アイコン / ▼ドロップダウン

## Lapsellへの応用メモ
- **マスコット:** Lapsellにも地下アイドルをイメージしたキャラやアイコンを設けると親しみやすさが増す
- **「無料」「手数料」訴求:** 「初期費用0円」「手数料5%〜」を黄色や対比色で大型表示する手法が有効
- **固定フッターCTA:** 「今すぐ出品する」ボタンを常時固定表示
- **タグ型機能説明:** 「3層自動録画」「即時配信」「手数料5%〜」をヒーロー内タグで簡潔表示
- **特典3カード:** 「練習が収益に」「ファンに届く」「手間ゼロ」の3カード（キャラ付き）を並べる構成
- **黄×ダーク系コントラスト:** アーティスト向けLP特有のかっこよさと、行動促進の黄色CTAを組み合わせられる
