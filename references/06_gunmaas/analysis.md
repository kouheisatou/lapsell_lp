## サービス・コンテンツ概要

「GunMaaS」は群馬県内の公共交通機関・タクシー・デマンド交通などをスマートフォン一つでシームレスに利用できるWEBサービス。「検索」「予約」「決済」機能をオールインワンで提供。

**主な機能:**
- リアルタイム経路検索（デマンド交通・シェアサイクル対応、遅延反映）
- 電子チケット（ICカードタッチ型 / スマホ提示型）
- デマンド交通予約（乗合バス）
- タクシー予約（前橋エリア限定）
- こどもデマンド（2025年3月末でサービス休止中）
- マイナンバーカード連携による特別割引
- JRE IDアカウントで会員登録

**電子チケット一覧（17種以上）:** ぐんまワンデーローカルパス / マイバス / 上毛電気鉄道 / わたらせ渓谷鐵道 / 関越交通バス / 敬老・若者割引パス 等

**運営:** 群馬県新モビリティサービス推進協議会 (Copyright © 2025)

## ターゲットユーザー

- **主要ターゲット:** 群馬県在住・訪問者（スマートフォン所持者）
- **サブターゲット:** 観光目的の来訪者（フリーパス系チケット）・高齢者（敬老割引パス）・若者（若者割引）・ファミリー（こどもデマンド）
- **特記:** スマートフォン非所持者向けにタッチ会員制度あり（係員サポートで交通系ICカードを登録可能）

## 特徴的なデザイン要素・テクニック

1. **機能説明の統一フォーマット:** 全機能（Route Search / Electronic Ticket / On-demand / Taxi）が同じ構造（英語タイトル+日本語サブ / 説明文 / ①②③使い方 / スクリーンショット3枚）で統一されている。
2. **ステップ番号の表示:** ①②③ という丸数字を段落冒頭に置き、手順を明確に区切る。視覚的に分かりやすい。
3. **チケットグリッド:** 17種以上のチケットを4列のカードグリッドで整理。タグ（「交通系ICカードタッチ」「スマホ提示」）でフィルタリングの示唆。
4. **英語見出し + 日本語解説の二重構造:** ブランディングと実用性を両立。
5. **フッターのオレンジ:** ヘッダーの白と対比的なオレンジ（#F08300）のフッター。視覚的なまとまり感。
6. **スクロールUPボタン:** ページが長いため、右下に常時表示。
7. **デコレーション丸:** ページ左右端に配置した半透明の丸が「浮遊感」と「移動のリズム」を演出。
8. **Robotoフォント:** 国際的なGoogleフォントを採用。モダンで読みやすく、英語表記との相性が良い。

### 機能的な参考点

- **会員登録不要で一部使える** という導線設計（ハードル低い試用→登録へ誘導）はLapsellにも参考になる。「ファンはアカウントなしで閲覧、購入時のみ登録」など。
- **複数ターゲット（観光客/地元民/高齢者/若者）** を1LPに収める設計。Lapsellもアーティスト側とファン側の両方を1LPに収める設計。

### カラー・デザイン参考

- ビビッドグリーン（#089942）は「公共感・信頼感」を出す色。Lapsellは若年層向けなのでより明るい色調でもOK。
- フラットイラスト（行動する人物・交通機関）はサービスの具体的なユースケースを視覚化する手法として参考に。Lapsellも「練習する様子・録画される様子・ファンが見る様子」をフラットイラストで表現できる。
- ページ全体が「白」ベースで清潔感があり、機能説明系には適している。Lapsellも機能説明セクションは白ベース推奨。

### 注意点

- GunMaaSは行政・公共サービスのため、コピーが説明的・丁寧すぎる傾向。Lapsellはもっと感情的・ドキドキ感を前面に出したコピーが効果的。
- 英語+日本語の二重表示は情報量が増えるため、Lapsellで使う場合は見出しのみに限定する。
- チケットグリッドのような情報量の多いセクションはLP初期には不要。成熟後のユーザー説明ページ向き。
