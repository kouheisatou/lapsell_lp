# Lapsell ロゴ・コンセプト連想

## 0. サービスの核（出発点）
**「練習の“時間”を、先に売る。中身は運しだい（ガチャ）。届くのは世界に1つ。」**
加工なしの“日常・裏側”を、ガチャ型のデジタル動画グッズとして売買するマーケットプレイス。初期費用0円・手数料5%〜。

## 1. 名前から連想（Lapsell / ラプセル）
- **Lap** … ①周回・ラップタイム（時間の計測）②lap=ひざ・抱える＝大切に持つ ③lapse=（時間が）経過する・ふとした隙間の時間
- **sell** … 売る・売買・マーケット
- **capsule（カプセル）** … 名前の末尾の音＝-sell/-cell/-sule。ガチャの殻、カプセルトイ
- **cell** … 区間・セル（動画を10秒ごとに分割した1区間＝1セル）／細胞＝小さな最小単位
- → キーワード：**時間 × 売買 × カプセル × 区間(セル)**

## 2. サービスの“モノ・行為”から連想
- カプセル（青白のガチャの殻）／ガチャを回す・開ける・パカッと開く瞬間
- 時間：時計・タイマー・タイムスタンプ「19:47:30」・砂時計・ストップウォッチ・録画RECの赤丸
- 動画：再生▶ボタン・フィルムのコマ・タイムラインの区間バー・10秒の帯
- 練習・裏側：スタジオ・マイク・楽器・汗・ステージ袖
- 所有・真贋：印・スタンプ・封蝋・指紋・チェックマーク・透かし
- 届く・開封：通知・贈り物・宝箱を開ける高揚

## 3. 感情・価値から連想
- ワクワク（中身がわからない高揚）／一点もの＝特別・誇り
- 「自分が見つけて育てた」物語／無名の今がいつか宝物に
- あたたかい応援・推し活・つながり／健全で前向き（射幸心は煽らない）

## 4. 形のモチーフ候補（ロゴに落とせる形）
1. **カプセル＝時計**：カプセルの中に時計の針／カプセルの合わせ目が時計の12-6軸
2. **カプセル＝再生ボタン**：カプセルの中に▶（動画＝時間が動く）
3. **「L」＋カプセル**：頭文字Lがカプセルや時間の針を抱える（lap=抱える）
4. **開くカプセル**：上下に割れて光がこぼれる＝開封の瞬間（ガチャ）
5. **時間の区間(セル)**：10秒の帯／タイムラインの1コマを切り取る四角＝「世界に1つ」
6. **砂時計×カプセル**：時間が落ちて“モノ”に結晶化する
7. **吹き出し＋▶**：素の声・裏側が届く
8. **封印（シール/印）×再生**：所有・真贋＋動画

## 5. トンマナ / 配色（ロゴ用）
- ブランド世界観：所有・将来価値・一緒に育てる。青白のカプセル。
- 配色案：①ブルー〜シアン（カプセルの青／信頼・デジタル）②白＋ネイビー（清潔・確かさ）③差し色に黄やコーラル（ワクワク・人肌の温かさ）
- 形：丸み（カプセル・親しみ）／幾何でシンプル／小サイズでも潰れない（アプリアイコン前提）
- 余白を活かしたミニマル。線は均一。1〜2色で成立。

## 6. ★方針変更：カプセル／ガチャの“見た目”から離れる
カプセルの殻・ガチャ筐体・つまみは**使わない**。本質モチーフ＝**時間（時計/タイムスタンプ/区間バー）×動画（再生▶/フィルム）×一点物・所有（印/封/ダイヤ）×素の瞬間**で表現。「中身は届くまで分からない」は概念として残すが、形では出さない。配色はブルー〜シアン＋ネイビー基調＋少量の差し色。

## 7. ロゴ10案の方向性（icons/ に生成・非カプセル）
1. time_to_play … 時計の針が再生ボタン▶に変わる（時間→動画）
2. segment_clip … タイムラインから10秒の区間を1つ切り出す（世界に1つ）
3. L_timestamp … 頭文字L＋タイムスタンプ/コロン「L 19:47:30」を抽象化したモノグラム
4. play_seal … 再生▶を封印/印で囲む（所有・本物の証明）
5. one_frame … フィルムの1コマだけ色づく（一点物・所有）
6. rec_moment … RECの赤丸＋時計（その瞬間を記録して売る）
7. play_heart … 再生▶とハート（推しの素の時間・応援）
8. lap_ring … 周回(lap)＝円のタイムリング＋切れ目（経過した時間を切り取る）
9. stamp_unique … 「1/1」や唯一の印（世界に1つ・一点物）を幾何で
10. minimal_L_play … 極限ミニマルな「L」＋▶のアプリアイコン向けシンボル

各案：単色（白）背景／中央配置／余白多め／均一な線／1〜2色／小サイズでも潰れないアプリアイコン適性。**カプセル形状・ガチャ筐体は描かない。**
