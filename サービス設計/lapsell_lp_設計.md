# Lapsell｜販売者向けLP 構成案

> 目的：中小・個人の**アイドル／楽曲アーティスト**に「練習の時間を出品してみよう」と思ってもらい、無料登録（または問い合わせ）まで導く。
> 設計方針：①自然検索で見つけてもらう（SEO）→ ②課題に共感 → ③仕組みを理解 → ④「本当に売れる？」の疑問を根拠でつぶす → ⑤不安（事故・手間・費用）を消す → ⑥行動。

---

## 1. SEO設計（自然流入の入口）

### 想定検索ユーザー（販売者側）
無名〜中堅の地下アイドル／インディーズの楽曲アーティスト・バンド本人、その小規模事務所・マネージャー。「もっと稼ぎたい」「物販以外の収入がほしい」「ファンともっと近づきたい」と考えて検索する層。

### 主要キーワード（H1・タイトル・本文に自然に織り込む）
- 地下アイドル 収益化 / アイドル 収入 増やす / アイドル 稼ぐ 方法
- インディーズ アーティスト 収益化 / バンド マネタイズ
- ファングッズ 販売 / デジタルグッズ 販売
- 推し活 マネタイズ（運営側視点）
- 練習動画 収益化 / 配信 収益化 投げ銭以外

### ロングテール（FAQ・各セクションの見出しで拾う）
- 特典会以外 収益 / 物販 売れない 対策
- ファンクラブ 個人 開設 / ファンに直接販売 アプリ
- 地下アイドル 副収入 / インディーズ 物販 限界
- アイドル オフショット 販売 / 練習風景 売る

### メタ情報（案）
- **title**：練習の時間を収益に変える｜Lapsell（ラプセル）｜アイドル・アーティスト向けグッズ販売
- **meta description**：捨てていた練習・裏側の時間が、世界に1つのデジタルグッズになって売れる。初期費用0円・手数料5%〜。地下アイドル／インディーズアーティストの新しい収益源、Lapsell。
- **構造化データ**：`Service` schema（提供者・対象・料金）＋ FAQ schema（後述FAQをマークアップ）。

> ※ SEOは「中身の濃さ × 想定検索語の自然な使用」で効く。キーワードの詰め込みは逆効果なので、各セクションの見出し（H2/H3）に1語ずつ自然に入れる程度に留める。
