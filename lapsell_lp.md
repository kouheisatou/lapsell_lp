## 全体コンセプト

**プラットフォーム名**: Lapsell（ラプセル）

**コアメッセージ**: 音楽クリエイターの制作過程を世界に1つだけのデジタルグッズとして販売できるプラットフォーム

**ターゲット**:
- プライマリ: 音楽クリエイター（AI生成との差別化に悩む制作者）
- セカンダリ: クリエイターを応援したいファン

**デザインシステム**:
- プライマリカラー: `#d4a574`（ゴールド）
- 背景色: `#0a0a0f`（ダーク）
- テキスト色: `#e8e8ed`（ライトグレー）
- フォント:

  - 本文: Inter（サンセリフ）

### 2. 問題提起セクション（Problem Section）

**セクション見出し**:
```
生成AIの台頭で、
こんな悩みを抱えていませんか？
```

**問題カード 1**:
- アイコン: Clock（時計）
- タイトル: 「AIに数秒でできることに、何時間もかける意味があるのか」
- 説明: 「朝から晩まで創作に没頭し、細部にこだわって完成させた作品。でもAIなら数秒で「それっぽい」ものができてしまう。AIが量産した作品も、魂を込めて作った作品も、同じ土俵で評価される。「どれだけ時間をかけたか」「どんな想いで作ったか」は、誰も知らない。」

**問題カード 2**:
- アイコン: PlayCircle（再生ボタン）
- タイトル: 「制作過程を配信しているけど収益が出ない」
- 説明: 「動画配信プラットフォームで制作過程を配信しても、広告収益は微々たるもの。視聴者数が伸びず、収益化の条件すら満たせない。時間をかけて配信しても、報われない現実に疲弊している。」

**共感メッセージ**（セクション下部）:
```
でも、諦める必要はない

AI時代だからこそ、「人が時間をかけて作った証明」に価値が生まれる。
あなたの作品の制作過程そのものが、誰にも真似できない唯一無二のコンテンツになる。
```

### 3. ソリューションセクション（Solution Section）

**セクション見出し**:
```
Lapsellでは、作品の制作動画を
世界に1つだけのグッズとして販売できます
```

**サブテキスト**:
```
試行錯誤の過程をタイムラプス動画として記録し、完成した作品と紐づけて販売。
ファンはあなたの創作プロセスに触れられる、世界に1つだけのデジタルグッズを手に入れられます。
```

**サービスフロー図**（ServiceFlowDiagram コンポーネント）:
- ビジュアル: クリエイター → Lapsell → ファンのフロー

**利用ステップ**:

1. **制作（録画）**
   - アイコン: Video（ビデオカメラ）
   - 説明: 「Lapsellアプリで制作を開始。録画ボタンを押すだけで、制作過程が自動的にタイムラプス動画として記録され、クラウドにアップロードされます。」

2. **作品完成**
   - アイコン: CheckCircle（チェックマーク）
   - 説明: 「作品が完成したら、録画を停止。制作過程動画が自動的に10分単位に分割され、各区間が世界に1つだけのグッズとして準備完了します。」

3. **販売**
   - アイコン: ShoppingCart（ショッピングカート）
   - 説明: 「イベント会場やSNSでLapsellのQRコードを表示。ファンが購入すると、制作過程動画と支援証明NFTが自動発行されます。」

4. **2次流通**
   - アイコン: Repeat（リピート矢印）
   - 説明: 「購入したファンは、制作過程動画を他のファンに転売可能。転売時には、あなたに一定のロイヤリティが自動的に支払われます。」

### 7. 収益セクション（Revenue Section）

**セクション見出し**:
```
実際の収益性
```

**サブテキスト**:
```
制作時間そのものが収入に
```

**データビジュアライゼーション**:

**アンケート結果（円グラフ）**:
- タイトル: 「制作過程動画10分に対する支払い意向（N=393）」
- データ:
  - 3,001円以上: 15% （色: `#d4a574`）
  - 1,001~3,000円: 8% （色: `#b88a5f`）
  - 1~1,000円: 9% （色: `#9d7550`）
  - 購入しない: 68% （色: `#2a2a3e`）
- ハイライト: 支払意向あり 32%
- 注記: 「※ 独自アンケート調査（2025年）」

**収益試算**:
```
支払い意向のある支援者の期待支払額

10分あたり 880円 × 1時間 6枠 = 時給期待値 5,280円
```

**注意書き**:
```
※ 実際の収益はクリエイターのフォロワー数や作品の人気度によって変動します
```

## コピーライティングのトーン＆ボイス

- **トーン**: 共感的、理解を示す、希望を与える
- **ボイス**: 専門的だが親しみやすい、クリエイターの気持ちに寄り添う
- **言葉遣い**:
  - 「あなた」を主語にして直接語りかける
  - AI時代の課題を明確に指摘し、解決策を示す
  - 感情的な価値と実利的な価値をバランス良く提示

## アニメーション・インタラクション

- **スクロールアニメーション**: 各セクションがビューに入った時にフェードイン
